## 主要参数详细
> appInfo.get

参数：withDefault(boolean 是否包含默认值)

返回：

platform: 平台(android or ios) 默认：'unknown',

app: 应用名称(mncg or yglc) ，默认：'unknown',

version: 应用版本 默认：'unknown',

ak: ak信息 默认：'unknown',

userid: 用户ID 默认：'-1',

sessionid: sessionId 默认：'-1'

> getQueryString

参数：key

返回：key对应的value值
